# @doo-iconik/lit

[![npm version](https://img.shields.io/npm/v/@doo-iconik/lit.svg)](https://www.npmjs.com/package/@doo-iconik/lit)

Lit web component: 595 hand-drawn doodle-style SVG icons for Lit 3.

Part of [doo-iconik](https://github.com/ajentik/doo-iconik).

## Install

```bash
npm i @doo-iconik/lit
```

## Usage

```js
import '@doo-iconik/lit';
```

```html
<doo-iconik-lit name="heart" size="lg"></doo-iconik-lit>
<doo-iconik-lit name="star" size="md" spin></doo-iconik-lit>
<doo-iconik-lit name="arrow-right" flipHorizontal></doo-iconik-lit>
```

## Props

All components accept: `name`, `size`, `spin`, `pulse`, `bounce`, `flipHorizontal`, `flipVertical`, `variant`, `animation`.

See the [main documentation](https://github.com/ajentik/doo-iconik#props) for full prop details.
