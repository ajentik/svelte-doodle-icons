# Tree-Shaking / Per-Icon Imports Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Enable per-icon imports so bundlers can tree-shake unused icons, reducing the 212KB monolith to only the icons actually used.

**Architecture:** Extend `generate.mjs` to produce individual icon modules at `packages/core/src/icons/{name}.ts`, each exporting a single `IconData` object. Add an export map in `packages/core/package.json` for `@doo-iconik/core/icons/{name}`. The existing monolith `icon-data.ts` remains for consumers who want all icons. Consumers can then do:

```typescript
// All icons (existing — 212KB)
import { iconData } from '@doo-iconik/core';

// Single icon (new — ~200 bytes)
import { heart } from '@doo-iconik/core/icons/heart';
```

**Tech Stack:** Node.js (generate.mjs), TypeScript, package.json exports map

### Task 1: Generate individual icon modules

**Files:**
- Modify: `generate.mjs`

- [ ] **Step 1: Add per-icon file generation to generate.mjs**

After the existing icon-data.ts generation block, add:

```javascript
// Generate individual icon modules
const iconsDir = join(__dirname, 'packages', 'core', 'src', 'icons');
mkdirSync(iconsDir, { recursive: true });

for (const name of iconNames) {
  const icon = rawData[name];
  const camelName = name.replace(/-([a-z0-9])/g, (_, c) => c.toUpperCase());

  let content = "// Auto-generated by generate.mjs — do not edit manually\n";
  content += "import type { IconData } from '../icon-data.js';\n\n";
  content += `export const ${camelName}: IconData = ${JSON.stringify(icon, null, 2)};\n`;
  content += `export default ${camelName};\n`;

  writeFileSync(join(iconsDir, `${name}.ts`), content);
}

// Generate icons barrel index
const iconsIndex = iconNames.map(name => {
  const camelName = name.replace(/-([a-z0-9])/g, (_, c) => c.toUpperCase());
  return `export { ${camelName} } from './${name}.js';`;
}).join('\n');

writeFileSync(join(iconsDir, 'index.ts'), '// Auto-generated by generate.mjs — do not edit manually\n' + iconsIndex + '\n');

console.log('Generated ' + iconNames.length + ' individual icon modules in packages/core/src/icons/');
```
